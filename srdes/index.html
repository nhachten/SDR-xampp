About our project

This project demonstrates how SDR can be applied to local area networks (LAN) in order to maintain network connection in a changing environment.

SDR is a technology that replaces traditional analog hardware in a wireless communication system with high-speed digital logic that is controllable through a software interface.

Our work involved writing software for the GNU Radio open-source framework. Our software provides various modulation schemes and error correcting codes to ensure fast delivery of data. The software also allows for TCP/IP connectivity over the air.

Sustainability

  • Dynamic flexibility and upgradeability
  • Easy to install different software applications as required for signal transmission
  • Multiband, multimode radio systems able to conform to various standards
  • Lower capital costs for service providers means cheaper service charges for customers
  • Production and devlopment times of radio systems greatly reduced
